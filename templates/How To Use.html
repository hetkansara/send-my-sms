What is Dynamic CSV Message?

Dynamic CSV Message is a message generated from a csv file. You can create any csv file containing your data collumns. Each row from your file will be converted into a message and system will send that message to the given contact number.

Steps:

1. Create .csv file (You can download a sample file from here Sample csv)

In sample file there are three columns for the message and other is contact number on which the message to be sent. These three column headers will be taken as variables in the Message Template.


Sample Csv File

2. Create a Message Template consisting of your column variables

Column headers will be used as variables in your template message. In above sample csv there are three variables "Name", "Marks" and "Exam". (Variables are case-sensitive. So be careful while using variables in message). All variables are declared in curly braces '{variable-name}' (ex. {Name}).
Template message for above sample can be
"Hello {Name},
You got {Marks} marks in {Exam} exam"
above template will generate two outputs from the csv file
"Hello Steve,
You got 50 marks in Internal exam" (This message will be sent to number 1234567890)
"Hello John,
You got 40 marks in External exam" (This message will be sent to number 1234567809)
You can set any number of variables at a time. User is advised to not to change header for the contact number. If you do so, the system may miss behave.




3. Send Message

Go to the "Dynamic CSV Message" tab.
Upload your .csv file and set your Template Message in MessageBox.
Click on Send SMS Button.
Done. 😄
